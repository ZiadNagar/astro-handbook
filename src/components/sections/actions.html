Actions v4.15+

Actions are type-safe server functions you can call from the client — like tRPC but built into Astro. Great for form submissions, mutations, anything that needs a server round-trip.

Defining Actions

src/actions/index.ts
import { defineAction, z } from 'astro:actions';

export const server = {
  newsletter: defineAction({
    accept: 'form',  // or 'json'
    input: z.object({
      email: z.string().email(),
    }),
    handler: async ({ email }) => {
      await subscribeToNewsletter(email);
      return { success: true };
    },
  }),
};

Calling Actions from Client

// From a React component (island)
import { actions } from 'astro:actions';

const handleSubmit = async (e) => {
  const { data, error } = await actions.newsletter({ email: email });
  if (error) return setError(error.message);
  setSuccess(true);
};
💡 Form Progressive Enhancement
Actions support native HTML forms with zero JavaScript as a fallback. The form still works without JS, and hydrates into a client-side experience when JS is available.
