Integrations

Integrations extend Astro's capabilities — UI frameworks, styling, deployment adapters, analytics, sitemaps, and more.

Adding Integrations

# Official integrations — auto-configures everything
npx astro add react       # React + React DOM
npx astro add vue         # Vue 3
npx astro add svelte      # Svelte
npx astro add solid-js    # Solid
npx astro add tailwind    # Tailwind CSS
npx astro add mdx         # MDX support
npx astro add sitemap     # Auto-generates sitemap.xml
npx astro add node        # Node.js adapter (SSR)
npx astro add vercel      # Vercel adapter (SSR)

Using Multiple UI Frameworks

---
import ReactCounter from './ReactCounter.jsx';
import VueSearch from './VueSearch.vue';
import SvelteModal from './SvelteModal.svelte';
---

<!-- All work together on one page! -->
<ReactCounter client:load />
<VueSearch client:idle />
<SvelteModal client:visible />
