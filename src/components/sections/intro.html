What is Astro?

Astro is a web framework focused on content-driven websites — blogs, documentation, marketing pages, e-commerce. Its core philosophy: ship less JavaScript to the browser. HTML is rendered at build time, and interactive components (called Islands) are hydrated on-demand.

⚡ Zero JS by Default

Every page ships as static HTML. JS only loads when you opt in via client directives.

🏝️ Islands

Interactive UI components exist as isolated islands in an otherwise static sea.

🔌 Bring Your UI

Use React, Vue, Svelte, Solid, Preact — or just plain HTML. All in one project.
