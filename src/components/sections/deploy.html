19 · Deployment

Deployment

Static (No Adapter Needed)

If output: 'static', just run npm run build and deploy the dist/ folder anywhere.

Vercel

Zero-config. Connect repo, auto-detects Astro.

Netlify

Drag-drop dist/ or connect Git.

Cloudflare Pages

Build command: npm run build, output: dist.

GitHub Pages

Use @astrojs/github-pages action.

AWS S3 + CloudFront

Upload dist/ to S3, serve via CloudFront.

Any Static Host

NGINX, Apache, self-hosted — dist/ is pure HTML/CSS/JS.

SSR Adapters

# Install the adapter for your platform:
npx astro add vercel      # Vercel Functions
npx astro add netlify     # Netlify Functions
npx astro add cloudflare  # Cloudflare Workers
npx astro add node        # Node.js (self-hosted)
npx astro add deno        # Deno Deploy

Docker (Node Adapter)

Dockerfile
FROM node:20-alpine
WORKDIR /app
COPY package*.json ./
RUN npm ci
COPY . .
RUN npm run build
EXPOSE 4321
CMD ["node", "./dist/server/entry.mjs"]
