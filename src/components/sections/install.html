Getting Started

Astro requires Node.js v18.17.1+. The fastest way to bootstrap a project is the official CLI wizard.

Run the create wizard

# npm
npm create astro@latest

# pnpm
pnpm create astro@latest

# yarn
yarn create astro

Answer the prompts

The wizard asks for: project name, template (empty / blog / portfolio), TypeScript preference, and package manager. Houston (Astro's mascot) guides you through.

Start the dev server

cd my-project
npm run dev
# → http://localhost:4321

Key CLI Commands

Command Action
npm run dev Start dev server with HMR
npm run build Build production output → dist/
npm run preview Preview production build locally
npx astro add react Auto-install integration
npx astro check Type-check project
npx astro info Debug environment info
