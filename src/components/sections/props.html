08 · Props & Slots

Props & Slots

Defining & Typing Props

---
interface Props {
  name: string;
  age?: number;          // optional
  variant: 'primary' | 'secondary';
}

const { name, age = 0, variant = 'primary' } = Astro.props;
---

<div class={`btn btn-${variant}`}>
  {name} - {age}
</div>

Astro.props vs Destructuring

// Both are equivalent:
const { title } = Astro.props;       // destructured
const title = Astro.props.title;   // direct access

Passing All HTML Attributes

---
const { class: className, ...rest } = Astro.props;
---
<button {...rest} class={`base-btn ${className}`}>
  <slot />
</button>

Slot Fallback Content

<slot>
  <!-- This renders if no slot content is provided -->
  <p>Default content here</p>
</slot>
