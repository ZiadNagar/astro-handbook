View Transitions

Astro has first-class support for the View Transitions API — smooth, app-like page transitions with minimal code.

Enable Globally

src/layouts/BaseLayout.astro
---
import { ViewTransitions } from 'astro:transitions';
---
<head>
  <ViewTransitions />   <!-- That's all! -->
</head>

Named Transitions

<!-- Page A -->
<img
  src="/hero.jpg"
  transition:name="hero-image"
  transition:animate="slide"
/>

<!-- Page B — same name creates a morph transition -->
<img
  src="/hero.jpg"
  transition:name="hero-image"
/>

Built-in Animations

Directive Effect
transition:animate="fade" Fade in/out (default)
transition:animate="slide" Slide left/right
transition:animate="none" No animation
transition:animate={myAnimation} Custom keyframe animation

Persist Elements Across Pages

<!-- Media player stays alive during navigation -->
<AudioPlayer transition:persist />

<!-- Keep scroll position of sidebar -->
<nav transition:persist="sidebar">...</nav>
